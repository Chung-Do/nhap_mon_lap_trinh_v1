Version 1.1 - April 7, 2026
───────────────────────────────────────────────────────────────

🐛 BUG FIXES:

  [CRITICAL] Fixed App Manager LIST_APPS not working
    • File: server/AppManager.java
    • Issue: Function returned empty list or encoding errors
    • Root causes:
      - Wrong encoding (UTF-8 instead of CP850 for Windows console)
      - Fragile CSV parsing that failed on some systems
      - Duplicate entries cluttering the output
      - Poor error messages giving no guidance

    • Solutions applied:
      ✓ Changed encoding to CP850 (Windows console encoding)
      ✓ Switched from CSV to TABLE format parsing
      ✓ Added deduplication (Set<String> to track unique apps)
      ✓ Filter out system processes (smss, csrss, svchost)
      ✓ Enhanced error messages with actionable suggestions
      ✓ Added debug logging with exit codes
      ✓ Improved output formatting

    • Impact: HIGH - Major feature fully restored
    • Testing: Required on Windows 10/11

📈 IMPROVEMENTS:

  [App Manager] Enhanced startApp() function
    • Auto-add .exe extension if missing
    • Smart PATH search for common apps
    • Process verification (check if actually started)
    • Better error messages with examples
    • Wait time to verify startup

  [App Manager] Enhanced stopApp() function
    • Auto-add .exe extension if missing
    • Verify termination success
    • Clear success/failure messages
    • Better result parsing

  [Logging] Added comprehensive debug logging
    • All operations log to server console
    • Exit codes reported for troubleshooting
    • Detailed error stack traces
    • Operation timings logged

  [Output] Improved formatting
    • Cleaner column alignment
    • Limit to 30 most relevant apps (was 50)
    • System processes hidden by default
    • Total count displayed
    • Vietnamese text support

Version 1.0 - April 1, 2026 (Initial Release)
───────────────────────────────────────────────────────────────

✨ FEATURES IMPLEMENTED:

  [Core]
    • Client-Server architecture with socket communication
    • JSON-based protocol
    • Multi-threaded server (one thread per client)
    • GUI for both client and server
    • System tray integration

  [Feature 1] App Management
    • List running applications
    • Start new applications
    • Stop running applications
    ⚠️  Known issue: LIST_APPS not working reliably

  [Feature 2] Process Management
    • List all processes with CPU/memory
    • Start new processes
    • Kill processes by PID
    • Cross-platform support

  [Feature 3] Remote Desktop
    • Real-time screen capture
    • Mouse and keyboard control
    • Adjustable quality settings
    • Responsive UI

  [Feature 4] Webcam Streaming
    • Live webcam feed
    • Auto-download FFmpeg on first use
    • Multiple quality options
    • Frame rate control

  [Feature 5] File Transfer
    • List remote directories
    • Download files from server
    • Upload files to server
    • Progress indication

  [Feature 6] Keyboard Monitoring
    • Start/stop keylogger
    • Real-time key capture
    • Retrieve logged data
    • Timestamp each keystroke

  [Feature 7] Network Diagnostics
    • Display IP addresses
    • MAC address information
    • Active connections list
    • Network adapter info

  [Feature 8] Power Management
    • Shutdown remote computer
    • Restart system
    • Logout current user
    • Safety confirmation
