CACH 1: Download tu Gyan.dev (De nhat - Essentials Build ~70MB)
========================================================================
1. Truy cap: https://www.gyan.dev/ffmpeg/builds/
2. Download file: "ffmpeg-release-essentials.zip"
3. Giai nen file ZIP

5. Copy "ffmpeg.exe" vao: resources/ffmpeg/
6. Chay: build_server.bat
7. Done! Server se bundle ffmpeg, khong can download nua.
