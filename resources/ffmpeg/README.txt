HUONG DAN: Dat file ffmpeg.exe vao day

Buoc 1: Download ffmpeg cho Windows
  - Truy cap: https://www.gyan.dev/ffmpeg/builds/
  - Download: ffmpeg-release-essentials.zip
  - Hoac: https://github.com/BtbN/FFmpeg-Builds/releases

Buoc 2: Giai nen va copy
  - Giai nen file ZIP
  - Tim file ffmpeg.exe trong thu muc bin/
  - Copy ffmpeg.exe vao thu muc nay (resources/ffmpeg/)

Buoc 3: Build lai project
  - Chay: build_server.bat
  - File ffmpeg.exe se tu dong duoc copy vao bundle

Ket qua:
  - Server khong can download ffmpeg moi lan chay nua
  - Chuc nang webcam hoat dong ngay lap tuc
